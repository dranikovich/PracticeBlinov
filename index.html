Математические приложения

Выберите одно из приложений для работы с математическими вычислениями

Нахождение начала вектора

Вычисление координат начальной точки вектора по известным компонентам вектора и координатам конечной точки. Формула: A = B - a

Транспонирование матрицы

Преобразование матрицы путем замены строк на столбцы. Формула: Aᵀ[i][j] = A[j][i]. Автоматическое создание матрицы при изменении размерности.

Перейти к приложению

Частичная сумма ряда ln(1+x)

Вычисление частичной суммы ряда Тейлора для функции натурального логарифма. Ряд: ln(1+x) = x - x²/2 + x³/3 - x⁴/4 + ... (сходится при -1 < x ≤ 1)

Инструкция по использованию

1. Нахождение начала вектора:

Введите компоненты вектора и координаты конечной точки через запятую

2. Транспонирование матрицы:

Задайте размерность матрицы и введите значения элементов

3. Частичная сумма ряда:

Введите значение x в интервале (-1; 1] и количество членов ряда

О проекте

Набор математических приложений для образовательных целей

Включает три инструмента для работы с:

  • Векторными вычислениями
  • Операциями с матрицами
  • Вычислением рядов
